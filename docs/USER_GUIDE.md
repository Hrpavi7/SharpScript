# SharpScript Calculator User Guide

## Features
- Basic arithmetic, scientific functions, memory store/recall, unit conversion, history.
- Natural language: "add 3 and 5", "convert 1 km to m".
- Multi-language UI, keyboard shortcuts, settings persistence, syntax preview.

## Usage
- Open tools/calculator/index.html in a browser.
- Type expressions or natural phrases; press Compute.
- Use Store to save the last expression and Recall to reuse.
- Toggle language and theme in header.

## Shortcuts
- Enter: Compute
- Ctrl+M: Store
- Ctrl+R: Recall
- Ctrl+H: Toggle history
